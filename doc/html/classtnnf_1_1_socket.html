Detailed Description

This is an abstract class. All other type of sockets derive from this.

If you want to make a class derive from this one, you have to override this methods:

Constructor. Used for Listener socket accept() method. If you want to use this, you have to make the socket by POSIX calls (like socket() or accept()). You have to call bind() too, if you want to bind the socket to port. You have to be sure to provide an std::shared_ptr exactly that variable which returned by socket() method, because if the memory address change the other methods will not work. Example:

std::shared_ptr<int> sock = std::make_shared<int>(-1);
sockaddr_storage address;
sock = socket(PF_INET6, SOCK_STREAM, IPPROTOTCP)

Automatically sets the socket option SO_REUSEADDR to 1
